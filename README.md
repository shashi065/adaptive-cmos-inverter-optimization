# Adaptive CMOS Inverter Optimization

## Overview
This project presents a data-driven approach to optimize CMOS inverter performance using open-source tools.

## Key Features
- Automated parameter sweep (VDD, W/L, Load Cap)
- Power and delay extraction using ngspice
- Power-Delay Product (PDP) optimization
- Heatmap and 3D visualization

## Tools Used
- ngspice
- Python (NumPy, Matplotlib, Pandas)
- Jupyter Notebook

## Results
- Identified optimal design point minimizing PDP
- Visualized trade-offs between power and delay

## Reproducibility
All simulations are automated and reproducible using the provided notebook.
